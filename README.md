# RustEngine

Vulkan Engine written in Rust.

## Prerequisites
- [Rust](https://www.rust-lang.org/tools/install)

## Usage

### Build

```bash
cargo build
```

### Run

```bash
cargo run
```

### Test

```bash
cargo test
```

### Documentation

```bash
cargo doc
```

### Format

```bash
cargo fmt
```

### Lint

```bash
cargo clippy
```
